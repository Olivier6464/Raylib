# Mon Projet Zig

"Affiche un triangle en utilisant la lib raylib".

## Prérequis

Pour compiler et lancer ce projet, vous devez avoir installé :
* **Zig** : version `0.15.2`

## Installation et Utilisation

Clonez le dépôt puis utilisez la commande standard de Zig pour compiler et exécuter le projet immédiatement :

```bash
zig build run
